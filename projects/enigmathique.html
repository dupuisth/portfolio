Enigmathique est une plateforme ludo-éducative pour apprendre les mathématiques de manière ludique, à déstination des collégiens. Le projet a été réalisé dans le cadre d'un projet académique au sein de l'IUT2 de Grenoble. Les enseignants créé un compte et renseignent les élèves dans la plateforme. Ils peuvent ensuite créer des salles de jeux, où les élèves peuvent se connecter à l'aide d'un code PIN, former des équipes et resoudre ensemble des salles d'escape game. Pendant la partie, l'enseignant peut suivre la progression des élèves via un tableau de bord pour voir les réponses, le temps restant, le nombre d'indices etc. ce qui lui permet de venir en aide aux élèves si besoin. Le projet est réalisé en React pour le front-end et ExpessJs pour le back-end.
